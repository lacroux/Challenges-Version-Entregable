🔐 Encriptador de Texto - Challenge Alura LATAMEste es un proyecto de encriptación y desencriptación de texto desarrollado como parte del desafío del programa Alura Latam y Oracle Next Education (ONE). La aplicación permite transformar mensajes de texto mediante un sistema de llaves de sustitución simples.

🚀 FuncionalidadesEncriptar: Transforma el texto ingresado reemplazando las vocales por "llaves" específicas.Desencriptar: Revierte el proceso, devolviendo el texto original a partir de un mensaje encriptado.Copiar: Incluye un botón que permite copiar el resultado directamente al portapapeles para facilitar su uso.Diseño Responsivo: La interfaz se adapta a diferentes tamaños de pantalla (Desktop y Mobile).

🔑 Llaves de EncriptaciónEl sistema utiliza las siguientes reglas de sustitución:

Vocal       Llave
 a            ai  
 e            enter
 i            imes
 o            ober
 u            ufat

🛠️ Requisitos de UsoPara que el encriptador funcione correctamente, el usuario debe seguir estas reglas:

* Utilizar únicamente letras minúsculas.
* No utilizar acentos.
* No utilizar caracteres especiales.

💻 Tecnologías Utilizadas

HTML5: Estructura del contenido.
CSS3: Estilizado, diseño visual y Media Queries para responsividad.
JavaScript: Lógica de encriptación, manipulación del DOM y funciones de copiado.
